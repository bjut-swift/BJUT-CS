## 贡献方式

### if 只提供经验分享 :

请在仓库中的 `docs` 文件夹中的对应仓库中新建你的经验分享文档，文件名为 `届-专业-姓名(可选择网名)`，格式建议为 `.md` 以方便阅读。

### else :

直接在 README.md 中的表格对应位置填写你的信息和资源链接即可，二进制文件、笔记等资源推荐自己建仓库存储，同时也欢迎到 [北京工业大学课程资源](https://github.com/bjut-swift/BJUT-Helper) 参与贡献。
